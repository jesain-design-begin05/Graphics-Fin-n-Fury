# Graphics-Fin-n-Fury

Title: Fin & Fury

Description: Fin & Fury is a 2D web-based underwater survival arcade game inspired by fish-eat fish mechanics. Once friends, Fin the small fish and Fury the poisonous fish have grown apart, turning into rivals in the underwater world. Players control Fin, who must grow by consuming smaller fish while avoiding larger predators and Fury’s attacks. To defend against Fury, players can collect pearls from clams to gain a temporary shooting ability. The game blends survival, strategy, and action, challenging players to navigate the dangers of the deep and face the one who was once a friend. Victory is achieved in each stage once all fish, including poisonous and giant poisonous fish, are cleared from the level.

Objectives:
* To develop a functional 2D underwater survival game for web browsers.
* To implement AI-controlled poisonous fish that actively chase the player.
* To design a pearl-based power-up system that grants a temporary shooting ability.
* To create a fish growth and progression mechanic that scales with player size.
* To integrate essential game systems, including scoring, collision detection, health, and game-over conditions.

Scope:
* Develop a 2D underwater environment for web browsers.
* Implement player movement, size progression, and attack mechanics.
* Include small fish, large fish, poisonous fish, and giant poisonous fish.
* Add pearl power-ups, score tracking, and level progression.
* Provide basic sound effects and user interface elements.
* Victory is determined by cleaning all fish in each stage, allowing progression to the next level.

Limitations:
* 2D graphics only, no 3D gameplay will be implemented.
* Single-player mode only; multiplayer is excluded.
* Limited environment variations and map designs.
* No advanced online features such as global leaderboards or cloud saving (initial version).
